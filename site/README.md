# FranchiseHub — локальна демонстраційна сторінка

Що в папці site:
- index.html — головна сторінка розрахунку стартових витрат.
- compare.html — сторінка порівняння трьох сценаріїв (Базовий / Розширений / Преміум).
- styles_Version1.css — стилі.
- script_Version1.js — загальна логіка (розрахунок, localStorage helpers).
- compare_Version2.js — логіка сторінки порівняння (presets, збереження сценаріїв).
- README.md — ця інструкція.

Як використовувати локально:
1. Помістіть папку site у корінь репозиторію або просто відкрийте site/index.html у браузері.
2. На index.html — зробіть розрахунок для одного сценарію.
3. На compare.html — порівняйте три сценарії, застосуйте пресети, збережіть сценарії у localStorage, завантажуйте їх у слоти.

Примітки:
- Збереження сценаріїв відбувається в localStorage браузера — дані зберігаються локально на вашому пристрої.
- Для подальшого поліпшення: додати UI для ручних multipliers, експорт/імпорт сценаріїв, експорт у CSV/PDF, серверну синхронізацію (auth).
