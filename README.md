# Сайт-франшизи — демоверсія (оновлено)

Що додано:
- Нова сторінка compare.html для порівняння трьох сценаріїв одночасно: "Базовий", "Розширений", "Преміум".
- Пресети сценаріїв (multipliers для ремонту, обладнання, маркетингу, зарплат).
- Можливість зберігати власні сценарії у localStorage (кнопки "Зберегти" поряд зі слотами).
- Список збережених сценаріїв із можливістю "Використати" (додати в один зі слотів) або "Видалити".

Файли:
- index.html — головна сторінка розрахунку.
- compare.html — сторінка порівняння сценаріїв (нова).
- styles.css — стилі (невеликі доповнення для сторінки порівняння).
- script.js — загальна логіка і функції (рахунок кошторису, localStorage).
- compare.js — специфічна логіка сторінки порівняння (presets, render, save/load).
- README.md — інструкція.

Як користуватися:
1. Відкрий index.html — базовий розрахунок для одного сценарію.
2. Відкрий compare.html — порівняй три сценарії одночасно.
   - Вибери франшизу/місто/площаду/штат.
   - Натисни пресет (або залиш за замовчуванням).
   - Назви сценарії або відредагуй назви.
   - Натисни "Порівняти" — отримаєш таблицю зі статтями витрат.
   - Щоб зберегти сценарій — натисни "Зберегти" поруч зі слотом. Збережені сценарії з'являться у списку.
   - "Використати" у списку дозволяє завантажити сценарій у один зі слотів (A/B/C).
   - Сценарії зберігаються у localStorage браузера (локально на вашому пристрої).

Можливі покращення:
- Додати UI для ручного налаштування multipliers (слайдери для кожної статті).
- Додати експорт/імпорт збережених сценаріїв (JSON, CSV, PDF).
- Додати більше пресетів або можливість клонути сценарій.
- Підвищити валідацію даних і красивіший UX при додаванні сценаріїв.
